# Expense Tracker

**Автор:** Иванов Мирослав.

## Описание программы
Приложение для учета личных финансов. Позволяет пользователю вносить данные о тратах, фильтровать их по категориям и автоматически рассчитывать общую сумму расходов. Данные сохраняются в формате JSON.

## Как использовать
1. Убедитесь, что у вас установлен Python 3.x.
2. Запустите файл: `python main.py`.
3. Введите сумму (число), выберите категорию и укажите дату.
4. Нажмите "Добавить расход".
5. Для фильтрации выберите нужную категорию в выпадающем списке и нажмите "Применить фильтр".

## Примеры использования
- **Тест 1 (Успех):** Ввод суммы "500", категории "Еда", даты "2023-10-25" -> Запись успешно добавлена.
- **Тест 2 (Ошибка):** Ввод суммы "-10" -> Программа покажет уведомление об ошибке.
